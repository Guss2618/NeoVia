# Design System NeoVIa

## Fundamentos

O NeoVIa deve ser mobile-first, com boa leitura em ambientes externos e elementos de toque confortaveis.

## Cores

- Azul principal: `#0f3f74`
- Azul escuro: `#0a2b52`
- Branco: `#ffffff`
- Texto secundario claro: `rgba(255, 255, 255, 0.78)`

## Interface

- Telas iniciais devem priorizar contraste alto, hierarquia clara e componentes centrais.
- Alvos de toque devem ter no minimo 44px de altura ou largura.
- Cards podem usar bordas arredondadas grandes para manter uma estetica amigavel e moderna.
